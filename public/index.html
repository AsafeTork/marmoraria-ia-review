Como funciona

Fluxo: Você escreve uma descrição → POST /api/gerar → DeepSeek gera código Three.js → servidor valida (mínimo 80 vértices, group não vazio) → até 3 tentativas automáticas → HTML com viewer 3D salvo no Supabase → exibido no iframe → você aprova ou rejeita.

Para outras IAs: Acesse GET /api/system para o schema completo de endpoints e como integrar. Os prompts ativos estão em GET /api/prompts e podem ser editados via POST /api/prompts sem restartar o servidor.

Melhorar o prompt: Vá em Prompts, copie o atual, edite com base nos rejeitos da galeria, salve nova versão. A próxima geração já usa o novo prompt.
